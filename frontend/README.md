# 🏝️ Nandhini Beach Resort Website

A full-stack resort booking website built using **React + Vite** for frontend and **Node.js + Express + MongoDB** for backend.

## 🚀 Features

### User Features
- 🏨 View resort rooms and facilities
- 🖼️ Gallery section
- ⭐ Customer reviews
- 📝 Submit reviews
- 🔐 User authentication
- 📱 Responsive design

### Admin Features
- 🔑 Admin login
- 📊 Admin dashboard
- 👥 Manage users
- ⭐ Approve customer reviews
- 📝 Manage resort content

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Bcrypt
